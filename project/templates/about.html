About iStats

iStats was created as a program to output statistics on sent iMessages. iStats was written in Python and makes use of the chat.db file generated by Mac devices. The front end was created using Flask, HTML, and JavaScript to ouput statistics in a readable format.

Division of Work

Bryce: Functions for top 10 most used words, characters, and emojis. (analysis.py)
Wyatt: Design and construction of website using Flask. Integration of data returned by python module with UI and JavaScript charts. (iStats.py)
Myra: Functions for max, min, average character count, most texted contact, and find a phrase. (analysis.py)
John: File read in and storing in pandas dataframe, various functions, data concerning dates and time. (gather.py)
